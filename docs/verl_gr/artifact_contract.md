# Artifact Contract

Phase A defines the artifacts exchanged between stages.

## Tokenizer Artifact

- tokenizer root
- representation type
- special token definition
- representation schema

## Stage Config Artifact

- task config
- stage config
- objective config

## Checkpoint Artifact

- SFT checkpoint
- Distill checkpoint
- RL checkpoint

## Reward and Decoding Artifact

- reward schema config
- decoding policy config

These artifacts are intended to become the stable handoff points between later phases.
